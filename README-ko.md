# Rico — 연구자를 위한 쉬운 홈페이지 제작 도구

![](/assets/demo-ko.webp)

[Rico](https://rico-v1.vercel.app/)는 코딩이나 디자인 지식이 없어도 **나만의 전문적인 홈페이지를 10분 안에 만들 수 있는 서비스**예요.

> https://rico-v1.vercel.app/

## 🌟 Rico는 왜 필요할까요?

연구자에게 홈페이지는 자신의 연구 성과와 활동을 알리는 중요한 얼굴이에요. 그런데 직접 만들려면 코드를 작성하거나 복잡한 툴을 익혀야 하는 경우가 많았죠.

Rico는 오직 **'연구자'** 에게만 집중해서, 꼭 필요한 항목들만 채워 넣으면 멋진 홈페이지가 완성되도록 만들었어요.

## ✨ 주요 기능

- **연구자 맞춤형 입력 양식** — 논문 목록, 소식(News), 프로젝트 이력 등 학술 활동에 꼭 맞는 항목들을 제공해요.
- **실시간 미리보기** — 내용을 입력하는 즉시 완성된 홈페이지 모습을 바로 확인할 수 있어요.
- **클릭 한 번으로 배포** — GitHub 계정만 있다면 `내아이디.github.io` 주소로 홈페이지가 전 세계에 공개돼요.
- **파일 직접 다운로드** — GitHub이 없어도 완성된 파일을 바로 내려받을 수 있어요.
- **영구 무료 & 간편한 수정** — 별도의 서버 비용 없이, 언제든 내용을 불러와서 수정할 수 있어요.
- **4가지 테마** — 4가지 테마 중 원하는 색상을 선택할 수 있어요.
- **한국어 / 영어 지원** — 두 가지 언어를 모두 지원해요.

## 🚀 시작하는 방법

1. **디자인 선택** — 마음에 드는 테마 색상을 골라요.
2. **프로필 작성** — 이름, 소속, 이메일 등 기본 정보를 입력해요.
3. **이력 채우기** — 진행한 프로젝트, 발표한 논문, 최근 소식을 추가해요.
4. **배포 또는 저장** — GitHub에 바로 배포하거나, 파일로 다운로드해요.

## 📋 입력할 수 있는 항목

| 섹션 | 항목 |
| --- | --- |
| 소개 (About) | 이름, 소속, 이메일, GitHub, LinkedIn, 블로그, X(Twitter), 프로필 사진, 자기소개 |
| 소식 (News) | 날짜 + 내용 |
| 논문 (Publications) | 제목, 저자, 학술지/컨퍼런스, 링크 |
| 프로젝트 (Projects) | 제목, 수행 기간, 참여 인원, 설명 |
| 커스텀 항목 | 직접 섹션 이름을 정해서 자유롭게 추가 (예: 학력, 수상 등) |
