# Rico — The Easiest Way for Researchers to Build a Homepage

![](/assets/demo-en.webp)

[Rico](https://rico-v1.vercel.app/) is a service that lets you **create your own professional homepage in 10 minutes — no coding or design skills needed**.

> https://rico-v1.vercel.app/

## 🌟 Why Rico?

A homepage is a researcher's face to the world — a place to share your work and accomplishments. But building one from scratch usually means writing code or learning complicated tools.

Rico is built exclusively for **researchers**, so you just fill in the fields that matter, and a great-looking homepage comes together on its own.

## ✨ Key Features

- **Researcher-tailored input forms** — Fields optimized for academic work, including publication lists, news, and project history.
- **Live preview** — See exactly what your homepage looks like as you type.
- **One-click deploy** — With a GitHub account, your homepage goes live at `yourid.github.io` instantly.
- **Direct file download** — No GitHub? No problem. Just download the finished files directly.
- **Free forever & easy to update** — No server costs, and you can reload your saved data to make changes anytime.
- **4 themes** — Pick your favorite color from 4 available themes.
- **Korean / English support** — Available in both languages.

## 🚀 How to Get Started

1. **Pick a design** — Choose a theme color you like.
2. **Fill in your profile** — Add your name, affiliation, email, and other basic info.
3. **Add your work** — Include your projects, publications, and recent news.
4. **Deploy or save** — Push directly to GitHub, or download as a file.

## 📋 What You Can Add

| Section | Fields |
| --- | --- |
| About | Name, affiliation, email, GitHub, LinkedIn, blog, X(Twitter), profile photo, bio |
| News | Date + content |
| Publications | Title, authors, journal/conference, link |
| Projects | Title, duration, team size, description |
| Custom sections | Name your own section and add whatever you'd like (e.g. Education, Awards) |
